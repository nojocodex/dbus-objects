Bases: object

This class represents a DBus object. It should be subclassed and to export DBus methods, you must define typed functions with the dbus_objects.object.dbus_object() decorator.

dbus_objects.object.dbus_method(interface=None, name=None, return_names=None, multiple_returns=False)¶

This decorator exports a function as a DBus method

The function must have type annotations, they will be used to resolve the method signature.

The function name will be used as the DBus method name unless otherwise specified in the arguments.

Parameters

  • interface (Optional[str]) – DBus interface name

  • name (Optional[str]) – DBus method name

  • return_names (Optional[Sequence[str]]) – Names of the return arguments

  • multiple_returns (bool) – Returns multiple parameters

Return type

Callable[[Callable[…, Any]], _DBusMethod]
