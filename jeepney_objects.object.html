jeepney_objects.object module¶

class jeepney_objects.object.DBusObject(name=None)¶

Bases: object

This class represents a DBus object. It should be subclassed and to export DBus methods, you must define typed functions with the jeepney_objects.object.dbus_object() decorator.

property dbus_name¶
Return type

str

get_dbus_methods()¶

Returns a dictionary of the DBus methods. The key holds the DBus method name and the value holds a reference to the function.

Return type

Dict[str, DBusMethod]

jeepney_objects.object.dbus_method(name=None)¶

Exports a function as a DBus method

The function must have type annotations, they will be used to resolve the method sigature.

The function name will be used as the DBus method name unless otherwise specified in the arguments.

Parameters

name (Optional[str]) – DBus method name

Return type

Callable[[Callable[…, Any]], DBusMethod]
